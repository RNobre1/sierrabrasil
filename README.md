# The Agent — Meteora Digital

Plataforma SaaS de agentes de IA conversacionais para atendimento via WhatsApp.

## Setup

```bash
cp .env.example .env
# Preencher .env com credenciais do Supabase
npm install
npm run dev
```

## Stack

React 18 + TypeScript + Vite | Supabase | OpenRouter | Evolution API | Apify
